== Description ==
Make your users login via link or QR code.
Attention: anyone who has the login link will be able to login as the user.

Background picture from; https://pixabay.com/en/qr-code-quick-response-code-scanning-1903447/
 

== Installation ==
Upload the zip file via FTP or Wordpress plugin installation page

1. Go to the user page, generate the QR and save it
2. QR will be displayed on the user profile page.
